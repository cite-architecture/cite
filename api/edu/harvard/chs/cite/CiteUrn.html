Package: edu.harvard.chs.cite

[Groovy] Class CiteUrn

    • edu.harvard.chs.cite.CiteUrn

    • A class representing a reference to a versioned object in a set of objects with shared properties, expressed in the notation of the CITE Object URN system. This class parses URNs expressed as Strings, and makes the components of the URN programattically accessible. Note that while the automatically generated groovydoc output does not show get* methods for all the CtsUrn's member properties, Groovy compilation automatically creates those public methods.
