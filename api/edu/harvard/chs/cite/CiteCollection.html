Package: edu.harvard.chs.cite

[Groovy] Class CiteCollection

    • edu.harvard.chs.cite.CiteCollection

    • A class representing the implementation of a CITE Collection in the context of a specific CITE Collection service. Abbreviations for extensions can be resolved with reference to the inventory documenting the CITE Collection Service where this collection is implemented.
